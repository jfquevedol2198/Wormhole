 Release 1.0.0 Red Dwarf

Features implemented

1. Profit and loss dashboard

  • Retrieving PnL data by selected portfolio
  • Visualising PnL on a line chart that allows zooming and panning by date
  • Downloading PnL reports generated from live data retrieved
  • Plotting moving median / moving averages for specified intervals
  • Viewing portfolio at any given time, selectable by form field, or by clicking on a line chart
  • Portfolio grouped by platform, and assets, currently generic view for all platforms
  • Browsing transactions for specified time period
  • Viewing operations for any transaction within transactions table
  • Summarising Profit and Loss for a given open and close dates

2. Portfolio management

  • Creating, modifying and deleting portfolio
  • Adding, removing, modifying addresses within portfolios

3. Market data management

  • Listing all exchange pairs within Wormhole
  • Listing all missing exchange pairs in Wormhole
  • Listing all missing method definitions in Wormhole
  • Ability to download those in CSV form
  • Option to remove all exchange rates from Wormhole
  • Basic filtering to only show active and/or requested

4. Configuration panel for Wormhole

  • Selecting connected Wormhole backend with option to connect to a different than default for the system
  • Listing of all exchange rate providers registered in system
  • Viewing and editing of shared settings
  • Health check display
  • Worker agents management (listing, renaming, disabling)
  • Worker tasks list
  • Worker tasks status
  • Ability to start, stop, or restart all workers
  • Schema information
  • Service status
  • Exceptions list

Features incomplete

  • Authentication / authorisation
  • No roles, everyone has access to everything
  • All blockchain features currently experimental
  • Display classes for portfolio view
  • Linking filter settings to url not done for all forms

Known issues

  • Error catching too aggressive sometimes
